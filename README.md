**Domain:** Environment, Climate & Sustainability

## Dataset

- **Title:** Personal Carbon Footprint Behavior Dataset
- **Source:** [Kaggle](https://www.kaggle.com/datasets/sonalshinde123/personal-carbon-footprint-behavior-dataset)
- **Description:** Daily behavioral data of 200 individuals across 7 days (1400 records), capturing transportation mode, energy usage, food type, screen time, waste generation, and eco-actions. The target is daily carbon footprint in kg CO₂ with a categorical impact level (Low/Medium/High).

## Setup

```bash
pip install -r requirements.txt
```

Place `personal_carbon_footprint_behavior.csv` in the project root and run the notebook sequentially.
